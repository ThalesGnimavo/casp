# 26-06-15-001 ‚Äî 0.4.0: the close loop, automated

## What shipped
- `casp ship <slug>` ‚Äî flips a phase's prompt to `status: shipped`, wires its
  `session_log` pointer, moves the slug `phases_queued` ‚Üí `phases_shipped`.
  Pure state mutation, never touches git.
- `casp close` ‚Äî bumps `last_commit`/`last_session_id` from HEAD + newest log,
  then runs check. Never commits.
- `casp check --all [root]` ‚Äî one report across every cockpit under a root.
- **Migrations are now fully opt-in** ‚Äî a project with no migrations dir gets no
  migration finding at all.

## Tests
- Ten new regression tests (ship, close never-commits invariant, opt-in
  migrations, `--all` aggregation). Suite green.
